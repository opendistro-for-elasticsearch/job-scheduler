## 2019-04-10, version 0.9.0.0

### Features

Initial release of JobScheduler plugin. Supports 2 type of schedules
* [Unix Cron](https://en.wikipedia.org/wiki/Cron) 
* Interval, defined by
  * start time
  * interval value
  * interval unit
